# LEAN4 Playground

Welcome to my LEAN4 Playground! Learning something new is always an exciting experience, and LEAN is a truly powerful tool. As someone wise said, "With great power comes great responsibility," and this project serves as my study and exploration playground for the LEAN4 theorem prover.

## Resources

I’m building this project using a variety of resources, including helpful tools, websites, and blog posts. Here are the key resources that have been invaluable throughout the study process:

1. [Aristotle - Harmonic Fun](http://aristotle.harmonic.fun)
2. [Functional Programming in LEAN](https://lean-lang.org/functional_programming_in_lean/)
3. [Theorem Proving in LEAN4](https://lean-lang.org/theorem_proving_in_lean4/)
4. [Mathematics in LEAN](https://leanprover-community.github.io/mathematics_in_lean/)
5. [Mathematical Resources at HHU](https://adam.math.hhu.de/)
6. [Terry Tao's Blog](https://terrytao.wordpress.com/)

## Acknowledgements

I want to sincerely thank all the researchers, professors, students, and community members who are contributing to the field of formal methods and theorem proving. Their work continues to inspire me and provides the foundation for this project.
